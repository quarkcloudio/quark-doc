# 前言

## 介绍
QuarkAdmin 是一个可以帮你快速搭建管理后台的工具；它提供的丰富组件，能帮助您使用很少的代码就能搭建出功能完善的管理后台。

## 系统特性

**内置功能**
* 管理员管理
* 用户管理
* 权限系统
* 菜单管理
* 系统配置
* 操作日志
* 附件管理

**内置组件**
* Layout组件
* Container组件
* Card组件
* Table组件
* Form组件
* Show组件
* ...

## 所需依赖
* laravel
* spatie/laravel-permission
* gregwar/captcha
* aliyuncs/oss-sdk-php
* guzzlehttp/guzzle
